# Cyberdeck

A modular, open cyberdeck platform by **Kali Assistant**
Maintained under **HyperPROBE Labs**

---

## Overview

Cyberdeck is an open hardware + software platform designed for:

* embedded Linux experimentation
* cybersecurity tooling
* portable lab environments
* RF / networking / low-level development

The project focuses on **modularity**, **repairability**, and **hackability**.

## Features

* Modular hardware architecture
* Multi-interface expansion (USB, Ethernet, RF, etc.)
* Embedded Linux support
* Designed for customization and field use
* Suitable for networking and cybersecurity workflows

---

## Philosophy

Cyberdeck follows a **practical open-source approach**:

* No vendor lock-in
* No unnecessary restrictions
* Designed for real-world use, not just demos

You are encouraged to:

* modify
* extend
* integrate into your own systems
